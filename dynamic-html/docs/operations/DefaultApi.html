This is the API

artistsGet

Returns a list of artists

URL

https://pokedexrn-api.herokuapp.com/artists

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    Limits the number of items on a page

  • Query: offset Integer

    Specifies the page number of the artists to be displayed

artistsPost

Lets a user post a new artist

URL

https://pokedexrn-api.herokuapp.com/artists

HTTP Method

POST

Response Type

Parameters

  • Body: body Artist(Artist)

artistsUsernameGet

Obtain information about an artist from his or her unique username

URL

https://pokedexrn-api.herokuapp.com/artists/{username}

HTTP Method

GET

Response Type

inline_response_200

Parameters

  • Path: username String
